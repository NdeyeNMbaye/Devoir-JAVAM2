# Achat en Ligne - Microservice Spring Boot
### Description
Application de gestion d'achats en ligne développée avec Spring Boot. Elle permet de gérer les utilisateurs, les produits, les achats et les ventes via une API RESTful documentée avec Swagger.

### Prérequis

Java 17
Maven 3.9
PostgreSQL
Redis
IntelliJ IDEA Ultimate

### Stack Technique

- Spring Boot 3.3
- Spring Data JPA — gestion de la base de données
- PostgreSQL — base de données relationnelle
- MapStruct — mapping entités/DTOs
- Lombok — réduction du code boilerplate
- Springdoc OpenAPI (Swagger) — documentation de l'API
- Spring Cache + Redis — mise en cache des données
- SLF4J (Logback) — gestion des logs
